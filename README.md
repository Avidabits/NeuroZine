
##NeuroZine


### Descripcion

Este proyecto agrega distintos proyectos para la visualización y sonorizacion de ondas cerebrales. 

#### Caja de Pandora I y II
El primero de ellos, Caja de Pandora, explora el concepto desde processing, partiendo de las librerías de Eric Mika.
En este proyecto no estoy incluyendo la sonorización

Caja de Pandora I explora la visualizacion a través de metáforas físicas, un sistema de partículas que simula chisporroteos cerebrales

Caja de Pandora II explora la visualización a través de personajes abstractos que actuan dependiendo de parámetros obtenidos de las ondas. Una raya loca, un punto azul, una cortina de sopor, etc. 

Ambos proyectos se integran en el mismo proyecto SW de Processing:  CajaPandora.pde

#### Caja de Pandora III
Caja de Pandora III es un proyecto de sonificación cerebral usando OpenVibe y Pure data. 
La comunicación entre ambos programas es vía TCP y OSC. 
Para el caso de TCP se han desarrollado los external de Pd tcpOpenVibe y tcpOpenVibe~
Se incluyen los escenarios OpenVibe y los patchs Pure Data usados en la sonificacion

#### NeuroCine

Neuro Cine es un proyecto OpenFrameworks que integra la sonorización y la visualizacion.
